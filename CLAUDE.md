# CLAUDE.md

## Project Overview

**Coreline Partners** - Premium landing page for a high-end investment management firm. Creating 5 different design variations to present to client.

**Scope:** Logo animation + Hero section + Single CTA (contact). No backend.

**Tech Stack:** Next.js 15, React 19, Tailwind CSS v4, Motion (Framer Motion), TypeScript

## Quick Start

```bash
npm run dev      # Dev server (port 3000)
npm run build    # Static export to /out
```

## Brand

- **Primary Blue:** `#3a86ff`
- **Slate Blue:** `#415a77`
- **Logo:** Two-part interlocking "C" shape

## Structure

```
app/           # Next.js App Router
components/    # UI components
lib/           # Utils, constants
public/        # Static assets
plans/         # Feature plans
docs/solutions/ # Compound engineering docs
```
